`schemaui` е°Ҷ JSON Schema
ж–ҮжЎЈиҪ¬жҚўдёәз”ұ`ratatui`гҖҒ`crossterm`е’Ң`jsonschema`й©ұеҠЁзҡ„е®Ңе…ЁдәӨдә’ејҸзҡ„з»Ҳз«Ҝз”ЁжҲ·з•ҢйқўгҖӮ

иҜҘеә“и§Јжһҗдё°еҜҢзҡ„жЁЎејҸпјҲеөҢеҘ—йғЁеҲҶгҖҒ`$ref`гҖҒж•°з»„гҖҒй”®еҖјжҳ е°„гҖҒжЁЎејҸеұһжҖ§зӯүпјүпјҢе°Ҷе…¶иҪ¬жҚўдёәеҸҜеҜјиҲӘзҡ„иЎЁеҚ•ж ‘пјҢе°Ҷе…¶е‘ҲзҺ°дёәй”®зӣҳдјҳе…Ҳзҡ„зј–иҫ‘еҷЁпјҢе№¶еңЁжҜҸж¬Ўзј–иҫ‘еҗҺйӘҢиҜҒз»“жһңпјҢд»Ҙдҫҝз”ЁжҲ·еңЁдҝқеӯҳд№ӢеүҚе§Ӣз»ҲеҸҜд»ҘзңӢеҲ°е®Ңж•ҙзҡ„й”ҷиҜҜеҲ—иЎЁгҖӮ

## еҠҹиғҪдә®зӮ№

- **жЁЎејҸдҝқзңҹеәҰ** вҖ“
  `draft-07`пјҢеҢ…жӢ¬`$ref`гҖҒ`definitions`гҖҒгҖҒ`patternProperties`гҖҒжһҡдёҫгҖҒж•°еҖјиҢғеӣҙд»ҘеҸҠеөҢеҘ—зҡ„еҜ№иұЎ/ж•°з»„гҖӮ
- **йғЁеҲҶе’ҢиҰҶзӣ–еұӮ** вҖ“
  йЎ¶еұӮеұһжҖ§жҲҗдёәж №ж ҮзӯҫпјҢеөҢеҘ—еҜ№иұЎиў«еұ•е№ідёәйғЁеҲҶпјҢеӨҚжқӮиҠӮзӮ№пјҲеӨҚеҗҲдҪ“гҖҒй”®еҖјйӣҶеҗҲгҖҒж•°з»„жқЎзӣ®пјүжү“ејҖе…·жңүиҮӘиә«йӘҢиҜҒеҷЁзҡ„дё“з”ЁиҰҶзӣ–еұӮгҖӮ
- **еҚіж—¶йӘҢиҜҒ** вҖ“
  жҜҸж¬ЎжҢүй”®йғҪеҸҜд»Ҙи§ҰеҸ‘`jsonschema::Validator`пјҢжүҖжңүй”ҷиҜҜпјҲеӯ—ж®өдҪңз”Ёеҹҹ +
  е…ЁеұҖпјүйғҪиў«ж”¶йӣҶе№¶дёҖиө·жҳҫзӨәгҖӮ
- **еҸҜжҸ’жӢ”зҡ„ I/O** вҖ“ `io::input`еҸҜд»ҘеӨ„зҗҶ
  JSON/YAML/TOMLпјҲйҖҡиҝҮеҠҹиғҪж Үеҝ—пјүпјҢиҖҢ`io::output`еҸҜд»Ҙиҫ“еҮәеҲ°ж ҮеҮҶиҫ“еҮәе’Ң/жҲ–д»»дҪ•еҗҜз”Ёж јејҸзҡ„еӨҡдёӘж–Үд»¶гҖӮ
- **еҶ…зҪ® CLI** вҖ“
  `schemaui-cli`жҸҗдҫӣдәҶдёҺеә“зӣёеҗҢзҡ„жөҒзЁӢпјҢеҢ…жӢ¬еӨҡзӣ®ж Үиҫ“еҮәгҖҒstdin/еҶ…иҒ”и§„иҢғе’ҢиҒҡеҗҲиҜҠж–ӯгҖӮ

## еҝ«йҖҹејҖе§Ӣ

```toml
[dependencies]
schemaui = "0.7.0"
serde_json = "1"
```

```rust
use schemaui::prelude::*;
use serde_json::json;

fn main() -> color_eyre::Result<()> {
    color_eyre::install()?;
    let schema = json!({
        "$schema": "http://json-schema.org/draft-07/schema# ",
        "title": "жңҚеҠЎиҝҗиЎҢж—¶",
        "type": "object",
        "properties": {
            "metadata": {
                "type": "object",
                "properties": {
                    "serviceName": {"type": "string"},
                    "environment": {
                        "type": "string",
                        "enum": ["dev", "staging", "prod"]
                    }
                },
                "required": ["serviceName"]
            },
            "runtime": {
                "type": "object",
                "properties": {
                    "http": {
                        "type": "object",
                        "properties": {
                            "host": {"type": "string", "default": "0.0.0.0"},
                            "port": {"type": "integer", "minimum": 1024, "maximum": 65535}
                        }
                    }
                }
            }
        },
        "required": ["metadata", "runtime"]
    });

    let options = UiOptions::default();
    let ui = SchemaUI::new(schema)
        .with_title("SchemaUI жј”зӨә")
        .with_options(options.clone());
    let frontend = TuiFrontend { options };
    let value = ui.run_with_frontend(frontend)?;
    println!("{}", serde_json::to_string_pretty(&value)?);
    Ok(())
}
```

## иҫ“е…ҘдёҺиҫ“еҮәи®ҫи®Ў

- `io::input::parse_document_str`е°Ҷ
  JSON/YAML/TOMLпјҲйҖҡиҝҮ`serde_json`гҖҒ`serde_yaml`гҖҒ`toml`пјүиҪ¬жҚўдёә`serde_json::Value`гҖӮеҠҹиғҪж Үеҝ—пјҲ`json`гҖҒ`yaml`гҖҒ`toml`гҖҒ`all_formats`пјүдҝқжҢҒдҫқиө–йЎ№зІҫз®ҖгҖӮ
- `schema_from_data_value/str`д»Һжҙ»еҠЁй…ҚзҪ®дёӯжҺЁж–ӯжЁЎејҸпјҢжіЁе…ҘиҚүзЁҝ -07
  е…ғж•°жҚ®е’Ңй»ҳи®ӨеҖјпјҢд»Ҙдҫҝ UI еҠ иҪҪзҺ°жңүеҖјгҖӮ
- `schema_with_defaults`е°Ҷи§„иҢғжЁЎејҸдёҺз”ЁжҲ·ж•°жҚ®еҗҲе№¶пјҢйҖҡиҝҮ`properties`гҖҒ`patternProperties`гҖҒ`additionalProperties`гҖҒ`dependencies`гҖҒ`dependentSchemas`гҖҒж•°з»„е’Ң`$ref`зӣ®ж Үдј ж’ӯй»ҳи®ӨеҖјпјҢиҖҢдёҚдҝ®ж”№еҺҹе§Ӣж ‘гҖӮ
- `io::output::OutputOptions`е°ҒиЈ…дәҶеәҸеҲ—еҢ–ж јејҸгҖҒзҫҺи§Ӯ/зҙ§еҮ‘еҲҮжҚўд»ҘеҸҠ`OutputDestination::{Stdout, File}`зҡ„еҗ‘йҮҸгҖӮж”ҜжҢҒеӨҡдёӘзӣ®ж ҮпјӣеҶІзӘҒеңЁиҫ“еҮәеүҚиў«жҚ•иҺ·гҖӮ
- `SchemaUI::with_output`е°ҶиҝҷдәӣйҖүйЎ№йӣҶжҲҗеҲ°иҝҗиЎҢж—¶дёӯпјҢд»ҘдҫҝеңЁдјҡиҜқз»“жқҹеҗҺиҮӘеҠЁеҶҷе…ҘжңҖз»Ҳзҡ„`serde_json::Value`гҖӮ

## JSON Schema вҶ’ TUI жҳ е°„

`build_ui_ast` е…ҲжҠҠи§ЈжһҗеҗҺзҡ„жЁЎејҸи§„иҢғеҢ–дёә UI ASTпјҢеҶҚз”ұ `form_schema_from_ui_ast`
е°ҶжҜҸдёӘеӯҗж ‘жҳ е°„дёә `FormSection`/`FieldSchema`пјҡ

| жЁЎејҸеҠҹиғҪ                                                     | з»“жһңжҺ§д»¶                                                |
| ------------------------------------------------------------ | ------------------------------------------------------- |
| `type: string`, `integer`, `number`                          | еёҰжңүж•°еҖјдҝқжҠӨзҡ„еҶ…иҒ”ж–Үжң¬зј–иҫ‘еҷЁ                            |
| `type: boolean`                                              | еҲҮжҚў/еӨҚйҖүжЎҶ                                             |
| `enum`                                                       | еј№еҮәйҖүжӢ©еҷЁпјҲеҚ•йҖүжҲ–еӨҡйҖүз”ЁдәҺж•°з»„жһҡдёҫпјү                    |
| ж•°з»„                                                         | еҶ…иҒ”еҲ—иЎЁж‘ҳиҰҒ + жҜҸдёӘйЎ№зӣ®зҡ„иҰҶзӣ–еұӮзј–иҫ‘еҷЁ                   |
| `patternProperties`, `propertyNames`, `additionalProperties` | еёҰжңүжЁЎејҸж”ҜжҢҒйӘҢиҜҒзҡ„й”®еҖјзј–иҫ‘еҷЁ                            |
| `$ref`, `definitions`                                        | еңЁеёғеұҖеүҚи§Јжһҗпјӣиў«и§ҶдёәеҶ…иҒ”жЁЎејҸ                            |
| `oneOf` / `anyOf`                                            | еҸҳдҪ“йҖүжӢ©еҷЁ + иҰҶзӣ–еұӮиЎЁеҚ•пјҢе°Ҷйқһжҙ»еҠЁеҸҳдҪ“жҺ’йҷӨеңЁжңҖз»ҲиҙҹиҪҪд№ӢеӨ– |

ж №еҜ№иұЎз”ҹжҲҗж ҮзӯҫпјӣеөҢеҘ—еҜ№иұЎжҲҗдёәеёҰжңүйқўеҢ…еұ‘ж Үйўҳзҡ„йғЁеҲҶгҖӮжҜҸдёӘеӯ—ж®өи®°еҪ•е…¶ JSON
жҢҮй’ҲпјҲдҫӢеҰӮ`/runtime/http/port`пјүпјҢд»Ҙдҫҝз„ҰзӮ№з®ЎзҗҶе’ҢйӘҢиҜҒеҸҜд»ҘзІҫзЎ®жҳ е°„й”ҷиҜҜгҖӮ

## йӘҢиҜҒз”ҹе‘Ҫе‘Ёжңҹ

- `jsonschema::validator_for`еңЁ`SchemaUI::run`ејҖе§Ӣж—¶зј–иҜ‘е®Ңж•ҙжЁЎејҸдёҖж¬ЎгҖӮ

- иҰҶзӣ–еұӮпјҲеӨҚеҗҲеҸҳдҪ“гҖҒй”®еҖјжҳ е°„гҖҒеҲ—иЎЁжқЎзӣ®пјүдјҡж №жҚ®еҪ“еүҚжӯЈеңЁзј–иҫ‘зҡ„еӯҗжЁЎејҸеҗҜеҠЁиҮӘе·ұзҡ„йӘҢиҜҒеҷЁпјҢеӣ жӯӨй—®йўҳдјҡеңЁзҰ»ејҖиҰҶзӣ–еұӮд№ӢеүҚжө®еҮәж°ҙйқўгҖӮ

### еҝ«жҚ·й”®зі»з»ҹ

е°ҶжҜҸдёӘеҝ«жҚ·й”®ж”ҫе…Ҙ`keymap/default.keymap.json`дёӯпјҢд»ҘдҫҝиҝҗиЎҢж—¶йҖ»иҫ‘гҖҒеё®еҠ©иҰҶзӣ–еұӮе’ҢиҮӘеҠЁз”ҹжҲҗзҡ„
README еҝ«жҚ·й”®еҸӮиҖғйғҪдҪҝз”ЁеҚ•дёҖдҝЎжҒҜжәҗгҖӮ

- **ж јејҸ** вҖ“ жҜҸдёӘ JSON
  еҜ№иұЎеЈ°жҳҺдёҖдёӘ`id`гҖҒиӢұж–Ү`description`гҖҒеҸҢиҜӯ`descriptionZh`гҖҒ`contexts`пјҲд»»дҪ•`"default"`гҖҒ`"collection"`гҖҒ`"overlay"`гҖҒ`"help"`гҖҒ`"text"`гҖҒ`"numeric"`пјүпјҢдёҖдёӘ`action`еҢәеҲҶиҒ”еҗҲзұ»еһӢд»ҘеҸҠж–Үжң¬`combos`еҲ—иЎЁгҖӮдҫӢеҰӮпјҡ

  ```json
  {
    "id": "list.move.up",
    "description": "Move entry up",
    "descriptionZh": "жқЎзӣ®дёҠз§»",
    "contexts": ["collection", "overlay"],
    "action": { "kind": "ListMove", "delta": -1 },
    "combos": ["Ctrl+Up"]
  }
  ```

- **е®Ҹ + и§ЈжһҗеҷЁ** вҖ“ `app::keymap::keymap_source!()` `include_str!`s
  JSONпјҢ`once_cell::sync::Lazy`еңЁеҗҜеҠЁж—¶дёҖж¬Ўи§ЈжһҗпјҢе№¶е°ҶжҜҸдёӘз»„еҗҲй”®зј–иҜ‘дёә`KeyPattern`пјҲй”®з ҒгҖҒжүҖйңҖдҝ®йҘ°з¬ҰгҖҒзҫҺи§ӮжҳҫзӨәеӯ—з¬ҰдёІпјүгҖӮ
- **йӣҶжҲҗ** вҖ“
  `InputRouter::classify`е§”жүҳз»ҷ`keymap::classify_key`пјҢиҜҘеҮҪж•°иҝ”еӣһеөҢе…ҘеңЁ JSON
  дёӯзҡ„`KeyAction`гҖӮ`keymap::help_text`ж №жҚ®`KeymapContext`иҝҮж»Өз»‘е®ҡпјҢиҝһжҺҘз”ЁдәҺ`StatusLine`е’ҢиҰҶзӣ–еұӮиҜҙжҳҺзҡ„зүҮж®өгҖӮ
- **з”ҹжҲҗж–ҮжЎЈ** вҖ“ `build.rs`дјҡи§Јжһҗ `keymap/default.keymap.json`пјҢе№¶йҖҡиҝҮжҳҫејҸ HTML
  marker еҲ·ж–° `README.md` дёҺ `README.ZH.md` дёӯзҡ„еҝ«жҚ·й”®еқ—пјӣеӣ жӯӨеёёи§„ Cargo
  жһ„е»әе°ұдјҡи®©еҸҢиҜӯеҝ«жҚ·й”®еҸӮиҖғдёҺиҝҗиЎҢж—¶иЎҢдёәдҝқжҢҒеҗҢжӯҘгҖӮ
- **жү©еұ•** вҖ“ иҰҒж·»еҠ еҝ«жҚ·й”®пјҢзј–иҫ‘
  JSONпјҢйҖүжӢ©жҡҙйңІеё®еҠ©ж–Үжң¬зҡ„дёҠдёӢж–ҮпјҢе№¶еңЁеј•е…Ҙж–°зҡ„иҜӯд№үе‘Ҫд»Өж—¶еңЁ`KeyBindingMap`дёӯиҝһжҺҘз»“жһң`KeyAction`гҖӮ
